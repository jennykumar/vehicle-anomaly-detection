🚗 Vehicle Sensor Anomaly Detection

📌 Overview
This project detects anomalies in vehicle sensor data using time-series analysis. It identifies abnormal patterns that may indicate system faults or unusual behavior.

🎯 Objective
To analyze sensor data over time and detect anomalies using statistical methods.

🛠️ Tools & Technologies
Python
Pandas
Matplotlib

⚙️ Approach
Loaded time-series sensor data
Visualized data trends
Applied statistical thresholding (mean ± standard deviation)
Marked data points outside threshold as anomalies

📊 Results
Successfully identified abnormal data points
Visualized anomalies using line plots and highlighted points
Demonstrated basic anomaly detection in time-series data

🚀 Future Improvements
Use advanced models like Isolation Forest
Apply real automotive sensor datasets
Deploy as a real-time monitoring system
